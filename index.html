Voice Notes Privacy Policy

This Privacy Policy explains what information Voice Notes processes, how that information is used, and what choices users have.

Last updated: March 30, 2026
Primary support: support@Theprimecore.com
Secondary support: theprimecore8@gmail.com

Overview

Voice Notes helps users manage notes, reminders, shopping lists, calendar notes, attached images, and backup files directly on their device.

Information We Collect

  • No automatic personal user account, ads, or analytics collection is included in this build.
  • Normal note content stays on the user's device unless the user chooses to share, export, or import content.
  • If a user contacts Primecore for support, the provided email address and message may be used to reply.

Information Processed Inside the App

  • Note titles, note text, calendar notes, shopping-list items, reminder times, favorites, and self-destruct settings.
  • Attached images selected or imported by the user.
  • Backup files created, imported, restored, or shared by the user.
  • Voice input when the user taps the microphone button.
  • Password and recovery-PIN verification data for protected notes.

Information Stored on the Device

  • Notes and related app content are stored locally on the user's device.
  • Password and recovery-PIN values are not stored in plain text.
  • Backup files are stored only where the user chooses to save them.
  • Voice Notes does not provide a developer-operated cloud note-sync or note-storage service.

Permissions and Why They Are Used

  • Microphone and speech-recognition access are used for voice input.
  • Notifications, exact alarms, and boot-completed support are used for reminders and reminder restoration.
  • Internet access is limited and is not used for a developer-operated server for normal note sync or backup.

Sharing, Export, and Import

When a user explicitly chooses to share a note, export a backup, or import content from another source, the selected content leaves the app only for the destination chosen by the user.

What the App Does Not Automatically Collect

  • The app does not require users to create an account.
  • The app does not include ads or analytics SDKs in this build.
  • The app does not sell note content or share note content with advertising networks.

Data Retention and User Controls

  • Local note data stays on the device until the user edits or deletes it.
  • Exported backup files stay where the user saved them until the user removes them.
  • If the user clears app storage or uninstalls the app, local app data may be deleted by the operating system.
  • Users can create, edit, delete, protect, export, import, and remove note content at any time.

Children's Privacy

Voice Notes is not directed to children under 13, and no developer-operated account system is provided.
